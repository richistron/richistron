Usage
=====

## Install 

To install correctly this app you'll need **nodejs v0.10.18** or higher and **ruby 1.9.3** or higher.

## Dependencies

```
gem install  compass
npm install
bower install
```

## Unit testing

```
grunt test
```

## Build

```
grunt build
```

## Build, test and jshint

```
grunt
```

## Development server

```
grunt server
```

## Deploy to gh-pages

```
git subtree push --prefix dist origin gh-pages
```
